Розділ 1: Вступ

HTML (HyperText Markup Language) - це мова розмітки для створення веб-сторінок. Вона визначає структуру та зміст веб-сторінки, тоді як CSS (Cascading Style Sheets) використовується для оформлення.

Важливо: HTML документ складається з елементів, які позначаються тегами. Теги оточують зміст і надають йому певне значення.

Підрозділ 1.1: Історія HTML

HTML був створений Тімом Бернерсом-Лі у 1991 році. Ось основні віхи розвитку:

  • HTML 2.0 (1995)
  • HTML 3.2 (1997)
  • HTML 4.01 (1999)
  • XHTML (2000)
  • HTML5 (2014)

Розділ 2: Основна частина

Сучасний HTML5 включає багато нових елементів, таких як:

  1. - верхній колонтитул
  2. - навігаційні посилання
  3. - розділ документа
  4. - незалежний вміст
  5. - нижній колонтитул

Розділ 4: Висновки

HTML є основною технологією для створення веб-сторінок. Він постійно розвивається, щоб відповідати сучасним вимогам.

Основні переваги HTML5:

  • Покращена семантика
  • Підтримка мультимедіа (, )
  • Графіка та ефекти (, SVG)
  • API для складних веб-додатків
